# Urban GPT Privacy Policy

![Privacy First](https://img.shields.io/badge/privacy-aggregated%20data-green)

_A privacy policy for the Urban Institute District Analyst GPT integration._

> ⚠️ This integration does not process, store, or transmit any student-level or personally identifiable data.

**Effective date:** April 2026

---

## Overview
This GPT action connects to the Urban Institute's publicly available 
Education Data API to retrieve aggregated, non-personally identifiable 
education data. No personal data is collected, stored, or transmitted 
through this integration.

## Data Accessed
This action queries publicly available datasets including CCD school 
directory data, enrollment summaries, and SAIPE poverty estimates. All 
data is aggregated at the school or district level and contains no 
individually identifiable student or staff information.

## Data Storage
No data retrieved from the Urban Institute API is stored by this 
integration. Responses are returned directly to the ChatGPT session 
and are subject to OpenAI's data handling policies for ChatGPT 
Enterprise.

## FERPA Compliance
All data accessed through this integration is aggregated and 
publicly available. No student-level personally identifiable 
information (PII) is accessed, stored, or processed. This integration 
is designed to align with FERPA principles and student data privacy 
best practices.

## Third Party Services
This action accesses the Urban Institute Education Data Portal API 
under their Open Data Commons Attribution License.

For Urban Institute's data terms of use, visit:  
https://educationdata.urban.org/documentation
