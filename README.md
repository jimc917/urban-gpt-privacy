  <h1>Privacy Policy</h1>
  <p><strong>Effective date:</strong> April 2026</p>

  <h2>Overview</h2>
  <p>This GPT action connects to the Urban Institute's publicly available 
  Education Data API to retrieve aggregated, non-personally identifiable 
  education data. No personal data is collected, stored, or transmitted 
  through this integration.</p>

  <h2>Data Accessed</h2>
  <p>This action queries publicly available datasets including CCD school 
  directory data, enrollment summaries, and SAIPE poverty estimates. All 
  data is aggregated at the school or district level and contains no 
  individually identifiable student or staff information.</p>

  <h2>Data Storage</h2>
  <p>No data retrieved from the Urban Institute API is stored by this 
  integration. Responses are returned directly to the ChatGPT session 
  and are subject to OpenAI's data handling policies for ChatGPT 
  Enterprise.</p>

  <h2>FERPA Compliance</h2>
  <p>All data accessed through this integration is aggregated and 
  publicly available. No student-level personally identifiable 
  information (PII) is accessed, stored, or processed. This integration 
  is designed to be consistent with FERPA principles.</p>

  <h2>Third Party Services</h2>
  <p>This action accesses the Urban Institute Education Data Portal API 
  under their Open Data Commons Attribution License. For Urban 
  Institute's data terms of use, visit: 
  educationdata.urban.org/documentation</p>
